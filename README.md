# Brynlee Carter Memorial Golf Tournament Website

## Quick Deploy to Netlify (FREE)

### Option 1: Drag and Drop (Easiest - 2 minutes)

1. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Sign up for a free account (or log in)
3. Drag the entire folder containing these files into the drop zone
4. Done! You'll get a URL like: `yoursite.netlify.app`

### Option 2: Connect to Git (Recommended for updates)

1. Create a free GitHub account at [github.com](https://github.com)
2. Upload these files to a new repository
3. Go to [netlify.com](https://netlify.com) and click "Add new site"
4. Choose "Import an existing project"
5. Connect your GitHub account
6. Select your repository
7. Click "Deploy site"

### Custom Domain (Optional)

After deploying, you can add a custom domain like `brynleecarterfoundation.org`:

1. In Netlify dashboard, go to "Domain settings"
2. Click "Add custom domain"
3. Follow the instructions to point your domain to Netlify
